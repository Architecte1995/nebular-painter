# nebular-painter
A little smooth mobile drawing HTML5 

*Outil de peinture cosmique et fractale — zero dépendance, 100% tactile*

**Cathédrale 1995** — © 2026 Architecte1995 & Kimi K 2.6 | MIT License

## Stack
HTML5 Canvas + JavaScript vanilla. Offline-first. Mobile-first.

## Features
- Pinceau Fractal (Julia local)
- Rosace / Kaléido / Mandala (1-12 axes)
- Nébuleuse Particulaire
- Ink Bleed (Perlin 2D)
- Gradient Orbital
- Constellation auto
- Palette Cathédrale (8 couleurs sacrées)
- Import image
- Export PNG
